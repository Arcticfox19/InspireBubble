# 快速开始

## 启动项目

1. 确保已安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 在浏览器中打开显示的地址（通常是 `http://localhost:5173`）

## 测试步骤

### 1. 设置 API Key

在应用界面中：
- 找到"API Key 设置"部分
- 输入您的 Gemini API Key
- 点击"保存"

**获取 Gemini API Key：**
1. 访问 [Google AI Studio](https://makersuite.google.com/app/apikey)
2. 登录您的 Google 账户
3. 创建新的 API Key
4. 复制并粘贴到应用中

### 2. 输入测试内容

在"输入内容"区域输入或粘贴以下测试内容：

```
React 是一个用于构建用户界面的 JavaScript 库。它采用组件化开发方式，让代码更易维护和复用。使用 React Hooks 可以更简洁地管理组件状态。JSX 语法让编写 UI 代码更加直观。虚拟 DOM 机制提升了渲染性能。
```

### 3. 转换内容

1. 点击"转换为对话格式"按钮
2. 等待 AI 处理（会显示加载动画）
3. 查看右侧预览区域的转换结果

### 4. 查看结果

转换成功后，您应该看到：
- ✅ 一个小红书风格的标题
- ✅ 对话列表（用户问题和 AI 回答）

## 预期结果

- 标题：符合小红书风格的吸引人标题
- 对话：将输入内容拆分为多个问答对
- 格式：JSON 格式，包含 title 和 content 数组

## 故障排除

### 问题：API Key 错误
- 检查 API Key 是否正确
- 确认 API Key 已启用 Gemini API
- 检查是否有足够的配额

### 问题：转换失败
- 确认输入内容不为空
- 检查网络连接
- 查看错误提示信息

### 问题：开发服务器无法启动
- 确认 Node.js 版本 >= 18
- 运行 `npm install` 安装依赖
- 检查端口 5173 是否被占用
